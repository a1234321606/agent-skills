# Agent Skills Library

A collection of professional, zero-dependency, and highly secure skills designed for AI agents.

## Directory Structure

```text
agent-skills/
├── jira-pilot/          # Professional Jira pilot for tasks, development, and personal todos
└── (more skills...)
```

## Installation

To install a specific skill from this repository, clone the repo and add the skill directory to your agent's skills path.

## Skills Overview

| Skill Name | Description |
